# Motivation Backlog

지속적인 학습 동기를 만들기 위한 메커니즘 후보 목록. MVP에 다 넣지 않고 효과적인 3가지만 우선 구현한다.

## MVP에 포함

### 1. 연속 학습 일수 (Streak)
- 가장 검증된 메커니즘 (Duolingo, Quizlet 등에서 효과 입증)
- 구현: 사용자가 하루에 1개 이상 문제를 풀면 streak +1. 빠지면 0으로 리셋.
- 리스크 완화: "freeze" 토큰(주 1회 자동 부여) — 부담을 너무 강하게 만들지 않기 위함

### 2. 주간 목표 + 진행 게이지
- 사용자 본인이 분 단위 목표 설정 (기본 60분)
- 매주 월요일 00:00에 리셋
- UI에 진행 바 + 남은 분 표시

### 3. XP / 레벨
- 문제 풀이당 XP 지급 (정답=10, 오답=2, 보너스=난이도에 비례)
- 누적 XP에 따른 레벨 (제곱근 곡선)
- UI에 다음 레벨까지 남은 XP 표시

## 백로그 (효과 검증 후 도입 검토)

### 약점 정복 배지
- 특정 태그(예: `te_form`)에서 최근 30회 정답률 90%+ 달성 시 배지
- 효과 가설: 명시적 진전 신호로 인한 성취감
- 도입 시점: 사용자 데이터가 쌓여 태그별 통계가 의미 있을 때

### 친구 / 리더보드
- 주간 XP 순위
- 효과 가설: 사회적 동기, 비교 유인
- 리스크: 부정적 비교 / 압박. 옵트인으로 구현 필요.
- 도입 시점: DAU 100+ 이후

### 매일 짧은 회고
- 학습 끝에 1줄 회고 입력 (오늘 어려웠던 점)
- 회고를 모아서 주간 리포트로 보여줌
- 효과 가설: 메타인지 활성화

### 학습 캘린더 히트맵
- GitHub 식 잔디. 매일 학습량 시각화
- 효과 가설: 연속성에 대한 시각적 보상

### 푸시 알림 (웹 푸시)
- 평소 학습 시간대 30분 전 리마인더
- 효과 가설: 망각 방지
- 리스크: 알림 피로. 빈도 신중하게.

### 콘텐츠 기반 보상
- 일정 XP 달성 시 "스토리 잠금 해제" — 사용자 레벨에 맞는 짧은 일본어 만화/이야기 제공
- 효과 가설: 내재적 동기와 연결
- 의존: 콘텐츠 풀이 충분히 다양해진 후

### 학습 페이스 적응 메시지
- "최근 3일간 정답률이 올라갔어요" 같은 LLM 기반 짧은 코칭 멘트
- 효과 가설: 개인 코치 같은 느낌
- 도입 시점: 약점 통계가 정확해진 후 (Phase 4 이후)

## 평가 지표 (향후)

각 메커니즘 도입 시 다음을 측정:
- D1/D7/D30 retention
- 평균 일일 학습 분
- 일일 활성 사용자(DAU) 대비 streak 보유 비율
- 옵트아웃 빈도
